# blueutil

Command-line utility to control Bluetooth based on [`blueutil` by Frederik Seiffert](http://www.frederikseiffert.de/blueutil).

## Usage

- Print bluetooth status:

		$ blueutil status

- Turn bluetooth on:

		$ blueutil on

- Turn bluetooth off:

		$ blueutil off

- Restart bluetooth (e.g. Continuity stopped working)

		$ blueutil restart
